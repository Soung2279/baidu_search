对 [优妮](http://wpa.qq.com/msgrd?v=3&uin=3472054241&site=qq&menu=yes) 的 “百度一下” 功能的 粗糙复现。

~~懒人福音~~ 在群内发送搜索词后，bot发送搜索的链接
